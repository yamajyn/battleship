
・はじめに
  battleship.exeは、敵艦・敵機に照準を合わせ、撃破するゲームです。
  Windows7/8.1/10 で動作を確認しています。


・操作方法：

  1)照準
　　マウスの位置が照準になります。砲弾は重力の影響を受けるので、照準は高めに設定しましょう。

  2)発射
　　マウスの左クリックで主砲を、右クリックで対空砲を発射します。上方向の最大角度は45度です。

  3)艦載機
　　スペースキーを押すことで艦載機（複葉機）を飛ばすことができます。
　　発艦・着艦時以外の飛行中はWキーで上方向、Sキーで下方向に移動が、スペースキーで爆撃を行えます。
　　爆撃後は自艦に自動で戻り、爆弾を補充します。墜落しない限り何度でも発艦可能です。

 
・ルール：
　
　敵艦に着弾させる +10
　敵艦を撃破する　+100
　敵機を撃墜する　 +30

　爆撃を受けることでライフが１減ります。
　ライフが０になるか敵艦を撃破し損ねたらゲームオーバーです。
